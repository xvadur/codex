# Web

Static landing page + demo for **XVADUR AI Recepcia**.

## Local Preview

Simplest preview (macOS):

```bash
cd web
python3 -m http.server 5173
```

Then open `http://localhost:5173/`.

## Cloudflare Pages

Recommended settings:

- Framework preset: `None`
- Build command: *(empty)*
- Output directory: `web`
